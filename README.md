PROYECTO WEB  UTILIZANDO CRUD PARA LA CREACION DE UN CAMPUS UNIVERSITARIO


Lenguajes Utilizados
HTML5 (HyperText Markup Language):
HTML5 es el lenguaje estándar para crear y estructurar contenido en la web.
Utilízalo para definir la estructura de tus páginas, como encabezados, párrafos, listas y enlaces.
CSS (Cascading Style Sheets):
CSS se utiliza para aplicar estilos visuales a tus páginas HTML.
Personaliza la apariencia de tus elementos, como colores, fuentes, márgenes y tamaños.
JavaScript (JS):
JS es un lenguaje de programación que se ejecuta en el navegador del usuario.
Úsalo para agregar interactividad a tus páginas, como eventos de clic, animaciones y validación de formularios.
Base de Datos
MySQL:
MySQL es un sistema de gestión de bases de datos relacional.
Utilízalo para almacenar información sobre estudiantes, profesores, cursos y otros datos relevantes para tu proyecto educativo.
Diseño y Estilos
HTML5 y CSS:
HTML5 define la estructura de tus páginas, mientras que CSS aplica estilos visuales.
Crea diseños atractivos utilizando etiquetas HTML y personaliza la apariencia con CSS.
Backend
Java:
Utiliza Java para desarrollar la parte del servidor (backend) de tu aplicación.
Conecta tu base de datos MySQL, procesa solicitudes del cliente y gestiona la lógica de negocio.
